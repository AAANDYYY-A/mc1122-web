# 🟫 Minecraft 1.12.2 - Web Edition

纯 JavaScript + Three.js 实现的 Minecraft 1.12.2 网页版，**单 HTML 文件**即可运行。

> 基于 Bare Bones 材质包配色 · 支持桌面端与手机触屏

## 🚀 快速开始

直接浏览器打开 `index.html`，或启动本地服务器：

```bash
python3 -m http.server 8080
# 访问 http://localhost:8080
```

### 🎨 创造模式背包
- **全屏物品选择界面** — 创造模式下按 E/📦 打开，生存模式打开原版物品栏
- **12 个分类标签**：🧱建筑 | 🛋️装饰 | ⚡红石 | 🍎食物 | 🐮生物掉落 | ⚒️工具 | ⚔️武器 | 🛡️盔甲 | 🧪酿造 | 📦材料 | 🔍全部 | ⭐已存
- **物品搜索** — 支持中文/英文模糊搜索
- **无限拿取** — 左键拿一组（64个），右键拿1个
- **快捷栏配置** — 💾 保存当前9格配置 / 📥 加载已存配置
- **销毁槽** — 拖入物品即销毁
- **装备栏** — 盔甲+副手直接穿戴

### 💾 存档系统
- 类 NBT 结构（`level.dat` + `level.dat_old` 自动备份）
- IndexedDB 持久化 · 多世界支持
- **开始界面** — 单人游戏（继续存档）/ 新建存档（随机种子新世界）
- **暂停菜单** — 手动保存 / **自动保存开关**（每5分钟自动存，状态持久化）
- **手机一键保存** — 顶栏 💾 按钮
- 区块按需加载/卸载 · 离开视距自动写入存储

## 🛠 技术亮点

| 技术 | 说明 |
|------|------|
| **渲染** | Three.js r157 · 索引化合并 BufferGeometry · 双面半透明 |

| **碰撞** | AABB 玩家碰撞 · 树叶实体碰撞箱 |

## 🔧 开发

```bash
# 任何静态服务器即可
python3 -m http.server 8080
npx serve .
```

直接编辑 `index.html` — 单文件包含全部 HTML/CSS/JS。
